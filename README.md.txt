# 🩸 Blood Demand Prediction using Machine Learning

A simple ML project that predicts blood demand based on historical data.

## 📌 Features
- Predict blood units required
- Detect shortage conditions
- Simple CLI-based interaction

## 🧠 ML Model
- Linear Regression

## 📁 Dataset
Contains:
- Month
- Emergencies
- Previous demand
- Units required
